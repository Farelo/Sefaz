1 passo: importar os database para o mongo db utilizando o seguinte codigo no terminal 
 "mongorestore -d reciclopac <directory_backup> "
2 passo: rodar no terminal o comnado "mongod" para start o banco de dados 
3 passo: rodar o comando "npm install" no terminal dentro da pasta "smart-api" para obter as dependencias 
4 passo: rodar o servidor no terminal dentro da pasta "smart-api" utilizando o comando "nodemon app.js"
5 passo: rodar o front utilizando o comando "ng serve"
